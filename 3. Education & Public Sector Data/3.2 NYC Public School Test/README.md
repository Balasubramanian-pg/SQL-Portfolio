# Analysis of NYC Public School SAT Performance

## Project Summary
This project analyzes SAT performance data from New York City's public high schools to uncover insights into academic achievement patterns. It identifies top-performing schools, compares performance across boroughs, and highlights specific trends like the strong showing of specialized high schools and disparities in scores across the city, suggesting areas for further investigation into factors influencing educational equity.
