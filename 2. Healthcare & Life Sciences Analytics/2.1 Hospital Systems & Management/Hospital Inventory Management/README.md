# Hospital Inventory Management

## Project Summary
The Hospital Inventory Management project aims to design and implement a comprehensive database schema for tracking medical supplies, equipment, and their usage within healthcare providers. Leveraging Snowflake for its scalability and performance, this project will create a robust and scalable solution to optimize stock levels, reduce costs, and ensure the availability of critical medical items. It will cover requirements gathering, data modeling, database design, SQL query development for managing and retrieving inventory information, and strategies for data integrity and security, providing a foundational system for efficient inventory oversight.
